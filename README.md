# Random Task Generator

## Автор: [Каролина]
## Описание:
Приложение для генерации случайных задач с фильтрацией по типу (учёба, спорт, работа), сохранением истории в JSON и возможностью добавлять свои задачи.

## Возможности:
- Генерация случайной задачи из предопределённого списка.
- Добавление собственных задач с указанием типа.
- Фильтрация истории по типу задачи.
- Сохранение и загрузка истории из файла tasks.json.
- Проверка корректности ввода при добавлении задач.

## Пример использования:
1. Запустите main.py.
2. Нажмите «Сгенерировать задачу» — появится случайная задача.
3. Используйте фильтр для просмотра задач определённого типа.
4. Добавьте свою задачу через поле ввода.
5. Сохраните историю кнопкой «Сохранить историю».
6. При следующем запуске история загрузится автоматически.
